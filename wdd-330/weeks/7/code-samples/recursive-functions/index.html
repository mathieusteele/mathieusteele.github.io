Did you know there was once a mathematician named Lothar Collatz who came up with what is known as the 3n+1 conjecture or the collatz conjecture (it also has many other names). There has not been found any number that doesn't eventually end up at one if you apply this formula to it: If even, cut it in half. If odd, multiply by three and add one.
